05 — Projets

Projets Réalisés

🛡️
CyberShield AI
✅ Terminé

Plateforme complète de prédiction d'attaques réseau propulsée par l'IA. Détecte et classe 6 types d'attaques (DDoS, Brute Force, Port Scan, SQL Injection, Botnet) en temps réel avec un modèle Random Forest atteignant 97.4% de précision. Authentification sécurisée SHA-256, dashboard interactif et monitoring live.

Python Streamlit Scikit-learn Plotly Random Forest SHA-256
⚡ GitHub
